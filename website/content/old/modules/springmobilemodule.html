SpringMobileModule

Integrates Spring mobile with device specific view resolving in an Across web stack.

Minimum requirements

  • Across 5.x for 3.x
  • Across 2.x.x for 2.0.x
  • Across 1.1.2+ for 1.0.x
  • Java 8
